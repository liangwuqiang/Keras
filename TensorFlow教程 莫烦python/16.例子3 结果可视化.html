构建图形，用散点图描述真实数据之间的关系。 （注意：plt.ion()用于连续显示。）

# plot the real data
fig = plt.figure()
ax = fig.add_subplot(1,1,1)
ax.scatter(x_data, y_data)
plt.ion()#本次运行请注释，全局运行不要注释
plt.show()

for i in range(1000):
